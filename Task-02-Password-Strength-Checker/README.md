# Password Strength Checker

## Description
This project is a Password Strength Checker developed using Python.

## Features
- Checks password length
- Checks uppercase letters
- Checks lowercase letters
- Checks numbers
- Checks special characters
- Classifies password as Weak, Medium, or Strong
- Gives suggestions to improve password strength

## Technologies Used
- Python

## How to Run

```bash
python password_checker.py
```
